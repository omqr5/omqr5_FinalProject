# What does Metadata Mean?

Metadata is is a set of data describes and gives information about other data.  
It helps both computers and people understand, find, organize, and manage digital information.

Think of metadata as a label or context for raw information. Without metadata, files become harder to search, sort, or understand.

--- 

## Why Metadata Matters
Metadata helps with;

### **Organization**
- Sorts files by type, size, date created, etc.

### **Search**
- Lets you find photos, documents, emails, or songs quickly.

### **Data Management**
- Helps systems track large amounts of information.

### ** Security & Privacy **
- Identifies file ownership  
- Stores sensitive details (sometimes without you knowing)
 
## Types of Metadata
### **1. Descriptive Metadata**
Helps identify and find information.  
Examples: title, author, keywords, summary.

### **2. Structural Metadata**
Shows how components relate or are organized.  
Examples: chapters in a book, links between files.

### **3. Administrative Metadata**
Provides file management details.  
Examples: creation date, file format, permissions, history.

These three categories are used across many industries and often overlap.
